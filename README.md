# React Events

### [Live Site](https://react-events-app.fly.dev/)

## Description

React Events is an events centered social media app. Users can create their own events, signup to events, and follow other users.

## Features

<ul>
<li> Front end - React with TypeScript </li>
<li> Back end - .Net 6.0 API with C# </li>
<li> Database - PostgreSQL </li>
<li> Clean Architecture with the CQRS + Mediator pattern </li>
<li> Application published via Fly.io </li>
<li> Authentication using ASP.NET Core Identity for login and registration </li>
<li> Live chat with SignalR </li>
<li> Cloudinary integration for photo upload </li>
<li> Paging and filtering the events </li>
<li> Semantic UI Library </li>
</ul>
